Add Photos 處理流程修改

// 修改前：預先生成 dataURL window.logger.log('Add photos: Resizing new photo:', file.name, 'Number:', photoNumber); const resizedDataURL = await resizeImage(file); const resizedFile = { name: file.name, size: file.size, type: file.type, lastModified: file.lastModified || Date.now(), dataURL: resizedDataURL }; newPhotos.push(resizedFile); // 修改後：不預先生成 dataURL（與 Upload Photos Folder 相同） window.logger.log('Add photos: Adding new photo:', file.name, 'Number:', photoNumber); const newFile = { name: file.name, size: file.size, type: file.type, lastModified: file.lastModified || Date.now(), dataURL: '' // Will be generated in renderPhotos() like Upload Photos Folder }; newPhotos.push(newFile);

照片渲染方法修改

// 修改前：只渲染新照片 window.logger.log('Add photos: Starting renderNewPhotosOnly...'); const lazyObserver = initLazyLoading(); await renderNewPhotosOnly(newPhotos, lazyObserver); window.logger.log('Add photos: renderNewPhotosOnly completed'); // 修改後：渲染所有照片（與 Upload Photos Folder 相同） window.logger.log('Add photos: Starting renderPhotos...'); const lazyObserver = initLazyLoading(); await renderPhotos(allPhotos, lazyObserver); window.logger.log('Add photos: renderPhotos completed');

3. 統一的處理流程

// 統一的照片處理流程（Upload Photos Folder 和 Add Photos 都使用） 1. 用戶選擇照片 ↓ 2. 創建 File 對象（不預先生成 dataURL） ↓ 3. 添加到 allPhotos 數組 ↓ 4. 調用 renderPhotos(allPhotos, lazyObserver) ↓ 5. 在 renderPhotos() 中生成 dataURL ↓ 6. 渲染照片到頁面 ↓ 7. 調用 autoSaveAllPhotosToIndexedDB() ↓ 8. 調用 saveDataToStorage()
