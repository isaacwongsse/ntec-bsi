照片重複問題修復測試

🔍 問題分析

根據控制台日誌分析，發現以下問題：

  • 重複的數據載入: loadDataFromStorage() 被多次調用
  • 重複的照片處理: 同一個照片的錯誤出現多次
  • 多個載入點: 照片處理發生在多個地方
  • 缺少去重邏輯: 沒有檢查重複照片

1. 修復前後對比

問題 修復前 修復後 狀態
重複載入 loadDataFromStorage() 被多次調用 添加 isLoadingData 標誌防止重複載入 ✅ 已修復
照片重複 沒有去重邏輯 添加基於文件名的去重邏輯 ✅ 已修復
錯誤處理 同一個照片錯誤多次出現 防止重複處理同一照片 ✅ 已修復
載入狀態 沒有載入狀態管理 添加載入狀態檢查 ✅ 已修復

2. 修復的代碼

添加載入狀態檢查

// Load data from storage let isLoadingData = false; // 防止重複載入的標誌 async function loadDataFromStorage() { if (isLoadingData) { window.logger.log('loadDataFromStorage: Already loading data, skipping duplicate call'); return; } isLoadingData = true; window.logger.log('loadDataFromStorage: Starting to load data from storage...'); // ... 載入邏輯 ... // 重置載入狀態 isLoadingData = false; }

添加照片去重邏輯

// 去重照片（基於文件名） const uniquePhotos = []; const seenNames = new Set(); for (const photo of allPhotos) { if (!seenNames.has(photo.name)) { seenNames.add(photo.name); uniquePhotos.push(photo); } else { window.logger.log('Removing duplicate photo:', photo.name); } } allPhotos = uniquePhotos; window.allPhotos = allPhotos; // 確保 window.allPhotos 同步

修改自動載入邏輯

// 自動載入照片數據（如果沒有通過會話恢復載入） try { const saved = await window.storageAdapter.getItem('photoNumberExtractorData'); if (saved && (!allPhotos || allPhotos.length === 0) && !isLoadingData) { window.logger.log('Auto-loading photo data from storage...'); await loadDataFromStorage(); } else if (isLoadingData) { window.logger.log('Data is already being loaded, skipping auto-load'); } } catch (e) { window.logger.error('Failed to auto-load photo data:', e); }

💡 修復方案

  1. 添加載入狀態檢查: 使用 isLoadingData 標誌防止重複載入
  2. 照片去重邏輯: 基於文件名去除重複照片
  3. 載入狀態管理: 在載入完成後重置狀態
  4. 自動載入優化: 檢查載入狀態避免重複調用
  5. 詳細日誌: 添加去重統計信息

3. 預期效果

✅ 修復後的效果：
  • 頁面重新載入時不會出現重複照片
  • loadDataFromStorage() 只會被調用一次
  • 照片處理錯誤不會重複出現
  • 控制台日誌更清晰，顯示去重統計
  • 照片載入更穩定，不會短暫顯示後消失

4. 測試步驟

🧪 測試步驟：
  1. 重新載入 index.html 頁面
  2. 檢查控制台日誌，確認 loadDataFromStorage 只被調用一次
  3. 檢查照片預覽區域，確認沒有重複照片
  4. 查看去重統計日誌：DEBUG: Photo deduplication:
  5. 確認照片處理錯誤不會重複出現
  6. 檢查照片是否穩定顯示

5. 控制台日誌檢查

⚠️ 修復後應該看到的日誌：
  • loadDataFromStorage: Starting to load data from storage... - 只出現一次
  • DEBUG: Photo deduplication: {originalCount: X, uniqueCount: Y, duplicatesRemoved: Z}
  • Removing duplicate photo: [filename] - 如果有重複照片
  • Data is already being loaded, skipping auto-load - 防止重複載入
